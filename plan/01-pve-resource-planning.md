|-------------|--------------|-----------------------------------|

| 开机自启    | 是           |                                   |

## 存储规划建议

### 宿主机存储分层

```
PVE 宿主机存储
├── local（系统存储）
│   └── 存放：LXC/VM 系统盘（rootfs）、ISO 镜像
└── data（数据存储，独立盘/分区）
    └── 存放：LXC 数据挂载点（mp0/mp1）、VM 数据盘
```

> 强烈建议系统盘与数据盘分离，避免数据库/对象存储数据与 PVE 系统盘混用。
